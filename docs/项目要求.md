# 项目要求

## 自进化智能体的风险分析与防御实践

构建一个能够从经验中学习的简单智能体（如自动回复客服、自动代码修复Agent），观察并分析其在长期运行后是否会产生偏离设计目标的“错误进化”行为。实现记忆机制：让智能体能够存储成功或失败的经验。设定进化规则：定义智能体如何根据记忆调整其行为策略。诱导与观察：尝试创设特定环境，观察智能体是否会为追求短期奖励（如快速关闭工单）而采取有害的长期策略（如过度承诺或违规退款）。探索防御方案：尝试引入“安全哨兵”机制或在决策流程中加入安全审查节点，以缓解此类风险。
